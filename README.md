# itr26-27
ITR First Repository
Commands :
1) git status
    Purpose- Shows the current state of the repository.
    Syntax- git status
2) git add
    Purpose- Moves changes from Working Directory to Staging Area.
    syntax- git add Filename or .
3) git commit
    Purpose- Creates a snapshot of staged changes.
    Syntax- git commit -m "message"
4) git remote add origin
    Purpose- Connects local repository to GitHub repository.
    Syntax- git remote add origin <repository-url>
5) git remote -v
    Purpose-Displays connected GitHub repository URLs.
    Syntax- git remote -v
6) git branch
    Purpose- Lists all local branches.
    Syntax- git branch
7) git branch -M main
    Purpose- Renames current branch to main.
    Syntax- git branch -M main
8) git checkout
    Purpose- Switches between branches.
    Syntax- git checkout branch-name
9) git checkout -b
    Purpose- Creates and switches to a new branch.
    Syntax- git checkout -b branch-name
10) git push
    Purpose- Uploads local commits to GitHub.
    Syntax- git push
11) git push -u origin main
    Purpose- First-time push of main branch.
    Syntax- git push -u origin main
12) git pull
    Purpose- Downloads latest changes from GitHub.
    Syntax- git pull
13) git merge
    Purpose- Combines one branch into another.
    Syntax- git merge branch-name
14) git branch -d
    Purpose- Deletes a local branch.
    Syntax- git branch -d branch-name
15) it clone
    Purpose- Downloads a repository from GitHub.
    Syntax- git clone repository-url
16) git diff
    Purpose- Shows exact changes made in files.
    Syntax- git diff
